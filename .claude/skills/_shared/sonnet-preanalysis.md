# Sonnet 선행 분석 절차

대형 구현 스킬(`/impl-backend-go`, `/impl-frontend`, `/impl-fullstack`)에서 에이전트 위임 전에 수행하는 사전 코드 분석.

## 절차

### 1. 심볼 추출
요구사항에서 핵심 심볼(함수명, 타입명, 컴포넌트명)을 추출한다.

### 2. 키워드 사전 검증
추출한 심볼이 실제 코드에 존재하는지 Grep으로 확인한다.
- 존재하는 심볼만 Agent에 전달
- 자연어 금지 (구체적 함수명/타입명만)

### 3. 코드 분석 실행
`codebase-analyst` Agent로 관련 모듈 구조 분석을 위임한다.

### 4. 결과 검증
분석 결과에서 언급된 파일/함수 2~3개를 Grep으로 재확인한다.
- 80% 이상 정확 → 채택
- 미만 → 폐기 후 직접 탐색
