# 6. 기록

## 개요
사용자가 참여한 이벤트의 이력을 확인하는 페이지.

## 화면 구성

```
┌─────────────────────────────────┐
│  [Logo]                [Avatar] │
├─────────────────────────────────┤
│  내 이벤트 기록                  │
│                                 │
│  ┌─────────────────────────┐    │
│  │ 이벤트 제목               │    │
│  │ 인플루언서 이름            │    │
│  │ 참여일: 2026-03-15       │    │
│  │ 상태: 진행 중             │    │
│  └─────────────────────────┘    │
│                                 │
│  ┌─────────────────────────┐    │
│  │ 이벤트 제목               │    │
│  │ 인플루언서 이름            │    │
│  │ 참여일: 2026-03-10       │    │
│  │ 상태: 당첨 🎉            │    │
│  └─────────────────────────┘    │
│                                 │
│  ┌─────────────────────────┐    │
│  │ ...                     │    │
│  └─────────────────────────┘    │
│                                 │
├─────────────────────────────────┤
│  홈  │ 팔로워 │ 이벤트 │ 기록 │ 파트너 │
└─────────────────────────────────┘
```

## 기록 카드 구성

| 요소 | 설명 |
|------|------|
| 이벤트 제목 | 참여한 이벤트 이름 |
| 인플루언서 정보 | 이벤트를 생성한 인플루언서 이름 |
| 참여일 | 이벤트에 참여한 날짜 |
| 상태 | 진행 중 / 마감 / 당첨 / 미당첨 |

## 기능

| 기능 | 설명 |
|------|------|
| 카드 클릭 | 이벤트 상세 페이지로 이동 |
| 필터 | 상태별 필터 (전체 / 진행 중 / 당첨 / 미당첨) |
| 정렬 | 최신 참여순(기본) |
| 무한 스크롤 | 하단 도달 시 추가 로드 |
| 빈 상태 | 참여한 이벤트가 없을 때 → 이벤트 탭 탐색 유도 |

## 접근 권한
- 로그인 필수
- 비로그인 시 로그인 유도 화면
