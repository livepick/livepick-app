# 7. 파트너

## 개요
인플루언서(파트너)를 위한 전용 영역. 파트너 가입, 이벤트 생성 및 관리 기능 제공.

---

## 7-1. 파트너 가입

### 진입 조건
- 로그인 상태이나 파트너 미가입 사용자가 파트너 탭 진입 시

### 화면 구성

```
┌─────────────────────────────────┐
│  [Logo]                [Avatar] │
├─────────────────────────────────┤
│                                 │
│      파트너가 되어보세요!         │
│                                 │
│  이벤트를 만들고 팬들과           │
│  소통하세요.                     │
│                                 │
│  채널명: [____________]          │
│  채널 URL: [____________]       │
│  카테고리: [선택 ▼]              │
│  소개: [__________________]     │
│                                 │
│       [ 파트너 가입하기 ]         │
│                                 │
├─────────────────────────────────┤
│  홈  │ 팔로워 │ 이벤트 │ 기록 │ 파트너 │
└─────────────────────────────────┘
```

### 가입 필드

| 필드 | 필수 | 설명 |
|------|------|------|
| 채널명 | O | 인플루언서 활동명 / 채널명 |
| 채널 URL | O | 유튜브 등 대표 채널 링크 |
| 카테고리 | O | 활동 분야 (게임, 뷰티, 먹방, IT, 일상 등) |
| 소개 | X | 간단한 자기 소개 |
| 프로필 이미지 | X | 파트너 프로필 이미지 업로드 |

### 가입 후
- 파트너 탭 진입 시 파트너 대시보드로 이동
- 승인 절차: 즉시 승인 (추후 심사 도입 가능)

---

## 7-2. 파트너 대시보드

### 화면 구성

```
┌─────────────────────────────────┐
│  [Logo]                [Avatar] │
├─────────────────────────────────┤
│  내 파트너 대시보드               │
│                                 │
│  팔로워: 1,234명                 │
│  진행 중 이벤트: 3개             │
│  총 이벤트: 15개                 │
│                                 │
│       [ + 이벤트 생성 ]          │
│                                 │
│  ── 내 이벤트 ──────────────     │
│                                 │
│  ┌─────────────────────────┐    │
│  │ 이벤트 제목               │    │
│  │ 상태: 진행 중             │    │
│  │ 참여자: 456명             │    │
│  │ 기간: 03/20 ~ 03/30      │    │
│  │            [관리]         │    │
│  └─────────────────────────┘    │
│                                 │
│  ┌─────────────────────────┐    │
│  │ ...                     │    │
│  └─────────────────────────┘    │
│                                 │
├─────────────────────────────────┤
│  홈  │ 팔로워 │ 이벤트 │ 기록 │ 파트너 │
└─────────────────────────────────┘
```

### 대시보드 요약 정보

| 항목 | 설명 |
|------|------|
| 팔로워 수 | 나를 팔로잉하는 사용자 수 |
| 진행 중 이벤트 | 현재 진행 중인 이벤트 수 |
| 총 이벤트 수 | 생성한 전체 이벤트 수 |

### 이벤트 목록
- 내가 생성한 이벤트 리스트
- 상태별 필터: 전체 / 진행 중 / 예정 / 마감
- 관리 버튼 클릭 시 이벤트 관리 페이지로 이동

---

## 7-3. 이벤트 생성

### 화면 구성

```
┌─────────────────────────────────┐
│  [←]  이벤트 생성        [저장]  │
├─────────────────────────────────┤
│                                 │
│  이벤트 제목: [____________]     │
│                                 │
│  이벤트 이미지: [이미지 업로드]    │
│                                 │
│  이벤트 설명:                    │
│  [________________________]     │
│  [________________________]     │
│                                 │
│  시작일: [____-__-__ __:__]     │
│  종료일: [____-__-__ __:__]     │
│                                 │
│  참여 조건:                      │
│  [________________________]     │
│                                 │
│  당첨자 수: [___]명              │
│                                 │
│       [ 이벤트 생성하기 ]         │
│                                 │
├─────────────────────────────────┤
│  홈  │ 팔로워 │ 이벤트 │ 기록 │ 파트너 │
└─────────────────────────────────┘
```

### 생성 필드

| 필드 | 필수 | 설명 |
|------|------|------|
| 이벤트 제목 | O | 이벤트 이름 |
| 이벤트 이미지 | X | 대표 이미지 (미등록 시 기본 이미지) |
| 이벤트 설명 | O | 이벤트 상세 설명 |
| 시작일시 | O | 이벤트 시작 날짜/시간 |
| 종료일시 | O | 이벤트 종료 날짜/시간 |
| 참여 조건 | X | 참여에 필요한 조건 설명 |
| 당첨자 수 | O | 당첨 인원 |

---

## 7-4. 이벤트 관리

### 기능

| 기능 | 설명 |
|------|------|
| 이벤트 정보 수정 | 제목, 설명, 이미지, 기간 등 수정 |
| 참여자 목록 조회 | 이벤트에 참여한 사용자 목록 |
| 당첨자 선정 | 랜덤 추첨 또는 수동 선정 |
| 당첨 결과 발표 | 당첨자에게 결과 노출 |
| 이벤트 조기 마감 | 종료일 전 이벤트 수동 마감 |
| 이벤트 삭제 | 이벤트 삭제 (참여자 있을 경우 확인 필요) |

### 당첨자 선정 플로우
```
이벤트 마감 → 참여자 목록 확인 → 추첨(랜덤/수동) → 당첨 결과 발표
```
