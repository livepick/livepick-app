# 5. 이벤트

## 개요
이벤트 목록 페이지. 팔로워 탭과 전체 탭으로 구분.

## 화면 구성

```
┌─────────────────────────────────┐
│  [Logo]                [Avatar] │
├─────────────────────────────────┤
│  [  팔로워  ] [  전체  ]         │
├─────────────────────────────────┤
│                                 │
│  ┌─────────────────────────┐    │
│  │ [인플루언서 썸네일+이름]     │    │
│  │ 이벤트 제목               │    │
│  │ 이벤트 기간               │    │
│  │ 참여자 수 / 상태          │    │
│  └─────────────────────────┘    │
│                                 │
│  ┌─────────────────────────┐    │
│  │ ...                     │    │
│  └─────────────────────────┘    │
│                                 │
├─────────────────────────────────┤
│  홈  │ 팔로워 │ 이벤트 │ 기록 │ 파트너 │
└─────────────────────────────────┘
```

## 탭 구성

### 팔로워 탭
- 내가 팔로잉한 인플루언서가 생성한 이벤트만 표시
- 로그인 필수 (비로그인 시 로그인 유도)
- 팔로잉한 파트너가 없으면 빈 상태 + 홈 탐색 유도

### 전체 탭
- 플랫폼 전체 이벤트 목록
- 비로그인 상태에서도 조회 가능

## 이벤트 카드 구성

| 요소 | 설명 |
|------|------|
| 인플루언서 정보 | 프로필 이미지 + 닉네임 (작게) |
| 이벤트 제목 | 이벤트 이름 |
| 이벤트 이미지 | 이벤트 대표 이미지 (선택) |
| 이벤트 기간 | 시작일 ~ 종료일 |
| 상태 | 진행 중 / 마감 / 예정 |
| 참여자 수 | 현재까지 참여한 인원 수 |

## 기능

| 기능 | 설명 |
|------|------|
| 카드 클릭 | 이벤트 상세 페이지로 이동 |
| 필터 | 상태별 필터 (진행 중 / 마감 / 예정) |
| 정렬 | 최신순(기본), 마감임박순, 인기순 |
| 무한 스크롤 | 하단 도달 시 추가 로드 |

## 이벤트 상세 페이지

| 요소 | 설명 |
|------|------|
| 인플루언서 정보 | 프로필 + 닉네임 (클릭 시 프로필 이동) |
| 이벤트 제목 | 이벤트 이름 |
| 이벤트 이미지 | 대표 이미지 |
| 이벤트 설명 | 상세 설명 텍스트 |
| 이벤트 기간 | 시작일 ~ 종료일 |
| 참여 조건 | 참여에 필요한 조건 설명 |
| 참여 버튼 | 이벤트 참여 CTA |
| 참여자 수 | 현재 참여 인원 |
| 공유 | 이벤트 링크 공유 기능 |

- 참여 버튼: 로그인 필수. 이미 참여한 경우 "참여 완료" 상태 표시
- 마감된 이벤트: 참여 버튼 비활성화, 결과 확인 가능
