# 2. 페이지 정의

## 공통 레이아웃

### Header
```
┌─────────────────────────────────┐
│  [Logo]                [Avatar] │
└─────────────────────────────────┘
```

| 요소 | 설명 |
|------|------|
| Logo | 좌측 고정. 서비스 로고. 클릭 시 홈으로 이동 |
| Avatar | 우측 고정. 사용자 프로필 이미지. 클릭 시 프로필/설정 메뉴 |

- 비로그인 상태: Avatar 자리에 로그인 버튼 표시

### Footer (Bottom Navigation)

```
┌─────────────────────────────────────────┐
│  홈  │ 팔로워 │ 이벤트 │ 기록 │ 파트너  │
└─────────────────────────────────────────┘
```

| 탭 | 아이콘 | 설명 |
|----|--------|------|
| 홈 | Home | 인플루언서 탐색 (메인) |
| 팔로워 | People | 내가 팔로잉한 파트너 목록 |
| 이벤트 | Calendar | 이벤트 목록 (팔로워/전체 탭) |
| 기록 | History | 내가 참여한 이벤트 기록 |
| 파트너 | Star | 파트너 전용 영역 (가입/관리) |

- 현재 활성 탭은 색상/아이콘으로 구분
- 파트너 탭은 모든 사용자에게 노출 (파트너 미가입 시 가입 유도 화면)
