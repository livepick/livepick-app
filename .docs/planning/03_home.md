# 3. 홈

## 개요
서비스 메인 화면. 인플루언서(파트너)들을 탐색할 수 있는 그리드 카드 리스트.

## 화면 구성

```
┌─────────────────────────────────┐
│  [Logo]                [Avatar] │
├─────────────────────────────────┤
│                                 │
│  ┌──────┐  ┌──────┐  ┌──────┐  │
│  │ 썸네일 │  │ 썸네일 │  │ 썸네일 │  │
│  │      │  │      │  │      │  │
│  │ 이름  │  │ 이름  │  │ 이름  │  │
│  │ 팔로워 │  │ 팔로워 │  │ 팔로워 │  │
│  └──────┘  └──────┘  └──────┘  │
│                                 │
│  ┌──────┐  ┌──────┐  ┌──────┐  │
│  │      │  │      │  │      │  │
│  │ ...  │  │ ...  │  │ ...  │  │
│  └──────┘  └──────┘  └──────┘  │
│                                 │
├─────────────────────────────────┤
│  홈  │ 팔로워 │ 이벤트 │ 기록 │ 파트너 │
└─────────────────────────────────┘
```

## 인플루언서 카드 구성

| 요소 | 설명 |
|------|------|
| 썸네일/프로필 이미지 | 인플루언서 대표 이미지 |
| 닉네임 | 인플루언서 이름 |
| 팔로워 수 | 해당 인플루언서의 플랫폼 내 팔로워 수 |
| 진행 중 이벤트 수 | (선택) 현재 진행 중인 이벤트 개수 배지 |

## 기능

| 기능 | 설명 |
|------|------|
| 카드 클릭 | 인플루언서 프로필 페이지로 이동 |
| 무한 스크롤 | 하단 도달 시 추가 로드 |
| 정렬 | 추천순(기본), 인기순, 최신순 |
| 검색 | 상단 검색바를 통한 인플루언서 검색 (추후 확장) |

## 비로그인 상태
- 인플루언서 목록 조회 가능
- 카드 클릭 시 프로필 조회 가능
- 팔로우 시도 시 로그인 유도
