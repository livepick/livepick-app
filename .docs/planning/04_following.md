# 4. 팔로워 (내가 팔로잉한 파트너)

## 개요
사용자가 팔로잉한 인플루언서(파트너) 목록을 그리드 카드 형태로 표시.

## 화면 구성

```
┌─────────────────────────────────┐
│  [Logo]                [Avatar] │
├─────────────────────────────────┤
│  내가 팔로잉한 파트너             │
│                                 │
│  ┌──────┐  ┌──────┐  ┌──────┐  │
│  │ 썸네일 │  │ 썸네일 │  │ 썸네일 │  │
│  │      │  │      │  │      │  │
│  │ 이름  │  │ 이름  │  │ 이름  │  │
│  │[언팔로우]│ │[언팔로우]│ │[언팔로우]│ │
│  └──────┘  └──────┘  └──────┘  │
│                                 │
├─────────────────────────────────┤
│  홈  │ 팔로워 │ 이벤트 │ 기록 │ 파트너 │
└─────────────────────────────────┘
```

## 카드 구성

| 요소 | 설명 |
|------|------|
| 썸네일/프로필 이미지 | 인플루언서 대표 이미지 |
| 닉네임 | 인플루언서 이름 |
| 진행 중 이벤트 수 | 현재 진행 중인 이벤트 개수 |
| 언팔로우 버튼 | 팔로잉 해제 |

## 기능

| 기능 | 설명 |
|------|------|
| 카드 클릭 | 인플루언서 프로필 페이지로 이동 |
| 언팔로우 | 확인 다이얼로그 후 팔로잉 해제 |
| 빈 상태 | 팔로잉한 파트너가 없을 때 → 홈에서 탐색 유도 메시지 |

## 접근 권한
- 로그인 필수
- 비로그인 시 로그인 유도 화면
