# 1. 프로젝트 오버뷰

## 서비스명
LivePick

## 서비스 개요
인플루언서(유튜버)와 팬(팔로워)을 연결하는 이벤트 플랫폼.
인플루언서가 이벤트를 생성하고, 팔로워가 참여하는 구조.

## 핵심 가치
- **인플루언서**: 팬과의 직접적인 소통 채널 확보, 이벤트를 통한 팬 engagement 강화
- **팔로워(사용자)**: 좋아하는 인플루언서의 이벤트를 한곳에서 확인하고 참여

## 사용자 유형

| 유형 | 설명 |
|------|------|
| 일반 사용자 | 인플루언서를 팔로우하고 이벤트에 참여하는 사용자 |
| 파트너(인플루언서) | 이벤트를 생성·관리하는 인플루언서 |

## 핵심 플로우

```
[일반 사용자]
회원가입 → 인플루언서 탐색(홈) → 팔로우 → 이벤트 확인 → 이벤트 참여 → 기록 확인

[파트너]
회원가입 → 파트너 가입 → 이벤트 생성 → 이벤트 관리
```

## 플랫폼
- 모바일 웹 우선 (Mobile First)
- 웹 반응형 지원

## 기술 스택
